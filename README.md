# IS436lab4
Design and develop a professional personal portfolio website using the Bootstrap framework. Your portfolio should showcase your skillset, resume, projects, and learning outcomes from IS 436 while applying strong UI/UX design principles.

# Required Pages (minimum 4)
Home — Introduction & call-to-action
About Me — Bio, background, and learning journey
Portfolio / Projects — Showcase 3-5 projects with tech stack
Skills & Resume — Technical skills, tools, and expertise
Contact — Contact form or contact information

# UI/UX Design Principles to Apply
Professional color scheme and typography
Clear visual hierarchy and intuitive navigation
Responsive design (Mobile, Tablet, Desktop)
Consistent branding and layout throughout
Accessibility (alt text, color contrast, keyboard nav)
